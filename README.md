# TN Skills Hackathon - Government Scheme Mapper

## StartupTN Hackathon Project

AI-powered platform to map Tamil Nadu government schemes to eligible beneficiaries.

### Features
- ✅ Form-based eligibility matching
- ✅ NLP-powered conversational search
- ✅ Tamil + English support
- ✅ ML-based semantic matching
- ✅ 40+ Tamil Nadu government schemes

### Tech Stack
**Backend**: FastAPI, Python, Sentence Transformers, Scikit-learn
**Frontend**: Next.js, TypeScript, Tailwind CSS (in progress)
